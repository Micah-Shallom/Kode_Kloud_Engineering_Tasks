2. Checking File Permissions of the executable file


        sudo ls -ltr /tmp/xfusioncorp.sh

        sudo chmod o+rx /tmp/xfusioncorp.sh


4. Executing the executable file


        sudo sh /tmp/xfusioncorp.sh
